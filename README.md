# Android_WeatherAlarm
일정 시간마다 날씨를 푸시 메시지로 알려주는 날씨 알리미 애플리케이션

## Basic Information

기획 및 개발 기간 : 

1차. 2020.07.20 ~ 2020.08.07(일시정지)
* location 받아오기
* 비동기 처리 작업으로 코루틴 적용하기
* Retrofit으로 날씨 api 받아오기
* 받아온 api를 view에 보이도록 적용하기
* 추후 데이터베이스 저장/가져오기 기능을 위해 room Entry, Dao 생성하기

2차. (8/20 시작 예정)

기술 스택 :
* `Android Studio`
* (언어) `Kotlin`
* (비동기 처리) `CoRoutine`
* (네트워크) `Retrofit`
* (데이터베이스) `Room`
* (이미지처리) `Glide`
